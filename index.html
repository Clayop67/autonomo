Roadmap

Voici les grandes étapes prévues pour le projet Autonomo :

  • Création du site et dépôt GitHub
  • Organisation des documents et du manifeste
  • Développement du code de base des agents IA
  • Recrutement de contributeurs

Comment contribuer

Si tu veux aider le projet, tu peux :

  • Envoyer des idées et suggestions via le formulaire
  • Contribuer au code sur GitHub
  • Aider à rédiger ou améliorer les documents
  • Tester le site et les formulaires

Pour contribuer au code, consulte le fichier CONTRIBUTING.md.
